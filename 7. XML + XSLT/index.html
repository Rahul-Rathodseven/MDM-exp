📚 XML + XSLT Demonstration

This page demonstrates the integration of XML (eXtensible Markup Language) and XSLT (eXtensible Stylesheet Language Transformations). XML provides structured data, while XSLT transforms that data into formatted HTML for display.

Quick Links:
📄 View Raw XML 💾 Download XML

✨ Key Features

  • Structured Data: 10 classic books with title, author, price, year, genre, and page count
  • XSLT Transformation: Converts XML to beautifully formatted HTML table
  • Client-side Processing: JavaScript XSLT transformation in the browser
  • Server-side Display: Open books.xml directly in browser to see automatic XSLT rendering
  • Sorting: XSLT sorts books alphabetically by title
  • Styling: Responsive design with modern UI

<?xml-stylesheet type="text/xsl" href="books.xsl"?>
<catalog>
    <book id="1">
        <title>The Great Gatsby</title>
        <author>F. Scott Fitzgerald</author>
        <price currency="USD">10.99</price>
        <year>1925</year>
        <genre>Fiction</genre>
        <pages>180</pages>
    </book>
    <!-- More books... -->
</catalog>
<?xml version="1.0" encoding="UTF-8"?>
<xsl:stylesheet version="1.0">
    <xsl:template match="/">
        <html>
            <head>
                <title>Book Catalog</title>
                <style>/* CSS styling */</style>
            </head>
            <body>
                <table>
                    <xsl:for-each select="//book">
                        <tr>
                            <td>
                                <xsl:value-of select="title"/>
                            </td>
                            <td>
                                <xsl:value-of select="author"/>
                            </td>
                            <td>
                                <xsl:value-of select="price"/>
                            </td>
                        </tr>
                    </xsl:for-each>
                </table>
            </body>
        </html>
    </xsl:template>
</xsl:stylesheet>
