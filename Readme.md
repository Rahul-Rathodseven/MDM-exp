## 🚀 How to Run

1. Install XAMPP and start Apache & MySQL
2. Clone repo:

   ```
   git clone https://github.com/Rahul-Rathodseven/MDM-exp
   ```
3. Place project in `htdocs` (for PHP)
4. Run in browser:

   ```
   http://localhost/project-folder
   ```
